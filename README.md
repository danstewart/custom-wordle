# Custom Wordle

A [wordle](https://www.powerlanguage.co.uk/wordle/) clone where you can set a word and send it to others!  


### Develop Locally
```shell
# Runs on http://localhost:8000/src/
./ctl.sh --serve
```

### Build for production
```shell
# One of compilation of app for deployment
./ctl.sh --build
```
